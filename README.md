# Selenium Login Automation

## 📌 Project Overview

This project demonstrates automated testing of a login functionality using Selenium WebDriver.

It includes:

* Positive login test (valid credentials)
* Negative login test (invalid credentials)
* Use of explicit waits for reliable execution
* Robust locator strategies (ID, CSS selectors)

---

## 🛠️ Tech Stack

* Java
* Selenium WebDriver
* Maven

---

## 🚀 Features

* Automates login flow on SauceDemo website
* Validates successful login using URL check
* Validates error message for invalid login
* Uses explicit waits (`elementToBeClickable`, `textToBePresentInElementLocated`)
* Clean and readable test structure
